# 🧠 Azure Mind App

An AI-powered productivity assistant that helps users save time by summarizing notes, drafting emails, and capturing meeting insights. Built with Loveable AI for intelligent, efficient task management.

**[🚀 Live Demo](https://azure-mind-app.lovable.app/)**

---

## ✨ Features

### Task Manager
- ✅ Add, edit, and delete tasks
- ✅ Mark tasks as complete
- 🎯 Priority levels (High / Medium / Low)
- 📅 Due dates with deadline tracking
- 🏷️ Categories and advanced filtering

### AI-Powered Tools
- 📧 **Email Generator** — Draft professional emails instantly
- 📝 **Meeting Summarizer** — Capture key points from conversations
- 🔍 **Research Assistant** — Gather and organize information
- 💬 **Chat Assistant** — Interactive AI companion for quick answers

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn package manager
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Th3Yaowa/azure-mind-app.git
   cd azure-mind-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   - Navigate to `http://localhost:3000`

## 📚 Project Learnings

### Deployment Challenges & Solutions

| Challenge | Solution |
|-----------|----------|
| **404 Error** | Ran `npm install + npm run dev` to resolve module dependencies |
| **SEO Issues** | Fixed 7 SEO issues identified by Loveable's audit: updated site title, meta descriptions, added Organization/SoftwareApplication JSON-LD structured data, optimized robots.txt and sitemap.xml |
| **Site Metadata** | Updated project name and description in Loveable settings |
| **Cache Issues** | Verified in incognito mode to ensure clean loading without cache conflicts |

✅ **Result:** App now loads error-free with improved SEO performance.
